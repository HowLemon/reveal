Answer the same questions as listed in Exercise 3 for the rooted tree illustrated.
  1. e) Which vertex is the parent of h?
  2. f) Which vertices are siblings of o?
  3. g) Which vertices are ancestors of m?
  4. h) Which vertices are descendants of b?
  1. siblings(手足點): 指的是具有同一個parent的其他點
  2. ancestors(祖先點): 該節點的所有祖先節點(一直溯源到root為止，不包含本身)
  3. descendants(後代點): 該點的所有後代節點(到樹葉點為止)
  1. e) The parent of $h$ is the vertex adjacent to $h$ and above $h$, namely $d$.
  2. f) Vertex $o$ has only one sibling, namely $p$,which is the other child of $o$'s parent, $i$.
  3. g) The ancestors of $m$ are all the vertices on the unique simple path from $m$ back to the root, namely $g,b$, and $a$.
  4. h) The descendants of b are all the vertices that have b as an ancestors,namely $e,f,g,j,k,l$, and $m$.

問題2

page 806, chapter 11.2 Exercise 22

Given the coding scheme $a: 001$, $b: 0001$, $e: 1$, $r: 0000$, $s: 0100$, $t: 011$, $x: 01010$, find the word represented by
  1. $01110100011$
  2. $0001110000$
  3. $0100101010$
  4. $01100101010$

霍夫曼編碼(Huffman coding)─採用可變長度碼

  1. 所有的樹被當作葉節點開始，由下而上合併，確保編碼為無前綴碼。
  2. 由權重最小的樹開始合併，因此頻率低的符號編碼較長，頻率高的符號編碼較短。

  1. 011  1 0100 011

    The first three bits decode as $t$. The next bit decode as $e$. The next four bits decode as $s$.The last three bits decode as $t$. Thus the world is $test$.

  2. 0001 1 1 0000

    beer

  3. 0100 1 01010

    sex

  4. 011 001 01010

    tax

樹的拜訪方式口訣:

  1. preorder traversal(前序追蹤): 根左右
  2. Inorder traversal(中序追蹤): 左根右
  3. Postorder traversal(後序追蹤): 左右根

The only difference here is that some vertices have more than two children: after listing such a vertex, we list the vertices of its subtrees, in preorder, form left to right. The answer is $a,b,d,e,i,j,m,n,o,c,f,g,h,k,l,p$.

問題5

page 839, chapter 11.5 Exercise 2

Use Prim’s algorithm to find a minimum spanning tree for the given weighted graph.

Prim algorithm(greedy approach)

  1. Prim演算法從一個edge F的空集合、一個由隨意的點初始的Y集合、還有一個包含全部點的V集合
  2. 目前Y集合有{v1}，選擇一個在V-Y集合中離Y集合點最小權重的Edge
  3. 假設v2離Y很近，將v2加入Y集合{v1,v2}，該edge加入F集合
  4. 然後重複2-3步驟直到Y=V時結束

結論: Prim演算法的時間複雜度$O(n^2)$，所產生最小生成樹的方式為對點的貪婪。

  1. We start with the minimun weight edge $a,b$.
  2. The least weight edge incident to the tree constructed so far is edge $a,e$, with weight 2, so we add it to the tree.
  3. Next we add edge $d,e$, and then dege $c,d$.
  4. This completes the tree , whose total weight is 6.
