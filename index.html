Prove that $m^2 = n^2$ if and only if $m = n$ or $m = −n$.
Prove that $m^2 = n^2$ if and only if $m = n$ or $m = −n$.
For the "if" part, there are two cases.

If $m=n$, then of course $m^2 = n^2$.
If $m=-n$ , then $ m^2 = (-n)^2 = (-1)^2 n^2 = n^2 $
For the "only if" part, we suppose that $ m^2 = n^2 $.
$\Rightarrow m^2 - n^2 = (m+n)(m-n) = 0$

Putting everything in the left and factoring, we have $(m+n)(m-n)=0$. Now the only way that a product of two numbers can be zero is if one of them is zero. Therefore we conclude that either $m+n=0$, or else $m-n=0$, and our proof is complete.
